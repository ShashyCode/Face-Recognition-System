The source code for the project is divide into four files:
1. Face Detection: for custom dataset
2. Trainer: For training the model from dataset.
3. Face Recogniton: It's the file that uses the model and recognizes faces in the vidoe feed.
4. Front: It's a simple front-end design made using python's GUI(Tkinter).

The dataset is stored in the folder of the same name.
The .yml file of the model is stored in the folder named trainer.
